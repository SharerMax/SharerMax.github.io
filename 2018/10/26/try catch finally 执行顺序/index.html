try catch finally 执行顺序

Oct 26, 2018
Java异常捕获执行顺序

无异常发生

有finally

输出：

try block
finally block

返回值：

7

无finally

输出：

try block

返回值：

9

有异常发生

有finally

输出：

try block before
catch block
finally block

返回值：

7

无finally

输出：

try block before
catch block

返回值：

8

总结

  1. try 块先执行，若未发生异常，则执行 finally 块；若try 中有异常发生，则从异常处中断，执行catch块，最后再执行 finally 块；
  2. 三个块中的 return 以最后执行 return 的值为结果返回。
